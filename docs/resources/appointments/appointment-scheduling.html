Appointment Scheduling

  • View Appointment
  • Create Appointment
  • Edit Appointment
  • Cancel Appointment

View Appointment

  1. Navigate to calendar
  2. Click on the gear Icon
  3. Check all the provider and facility resources you want to show on the calendar

Create Appointment

  1. Navigate to calendars
  2. Change the calendar view to day, week or month
  3. Change the view to provider or resource
  4. Provider view calendar:
  5. Resource view calendar:
  6. Click on any empty time slot to schedule an appointment or reserve facility resource for patient
  7. Enter the required details and click on confirm button to save changes

Edit Appointment

Practice can edit an appointment if needed by following the steps mentioned

  1. Navigate to calendar
  2. Click on appointment which has to be edited
  3. Click "Edit" on "Appointment Details"
  4. MEdit the appointment
  5. Click on "Confirm" Button to save changes

Cancel Appointment

Practice can cancel an appointment if needed by following the steps mentioned

  1. Navigate to calendar
  2. Click on appointment which has to be cancelled
  3. Click "Cancel" on "Appointment Details"
  4. Mention reason for the cancellation of the appointment
  5. Click on "Confirm" Button to save changes
