Appointment Request

If an appointment request is sent from PHR or Practice Website, then it needs to be confirmed if the request is not pre confirmed. Refer Settings > Calendar > Online Appointments to check wether the appointment is set to be pre confirmed or not

  • Confirm a Request
  • Deny a Request
  • Delete a Request

Confirm a Request

  1. Navigate to calendar
  2. Click on Appointment Requests
  3. Click on "Confirm" link to proceed further
  4. Make desired changes and click on "Confirm" button to confirm the appointment

Deny a Request

  1. Navigate to calendar
  2. Click on Appointment Requests
  3. Click on "Deny" link to proceed further
  4. Mention the reason for denying the appointment
  5. Click on "Deny" button to confirm

Delete a Request

  1. Navigate to calendar
  2. Click on Appointment Requests
  3. Click on "Delete" link to proceed further
  4. Clock on OK button to delete an appointment
