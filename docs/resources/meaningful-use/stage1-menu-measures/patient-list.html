Menu 3 : Patient Lists

Measure Overview : Generate a report for listing patients with a specific diagnosis.

Exclusion : None

ChARM EHR Workflow : Follow the steps below to generate patient list based on diagnosis.

  1. Go to Analytics > Patient Statistics section
  2. Search and select the required diagnosis
  3. Click on the 'Generate' button.

For more information, refer to the CMS guideline for this measure
