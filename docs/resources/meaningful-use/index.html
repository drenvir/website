What is Meaningful Use?

Meaningful Use (MU) has been established by Centers for Medicare and Medicaid (CMS) to measure and improve patient care by using a certified EHR. The following are different status for meeting the meaningful use guidelines.

Meaningful Use consists of the following measures.

  1. Stage 1 Measures
    • Core Measures ( Eligibility Criteria : 13/13 )
    • Menu Measures ( Eligibility Criteria : 5/9 )
  2. Stage 2 Measures
    • Core Measures ( Eligibility Criteria : 17/17 )
    • Menu Measures ( Eligibility Criteria : 3/6 )
  3. Clinical Quality Measures
