Core 1 : CPOE for Medication, Lab and Radiology Orders

Measure Overview : Use CPOE (Computerized Physician Order Entry) to record medication, laboratory and radiology orders.

Medication Order

Denominator : Number of medication orders created by the provider during the EHR reporting period.

Numerator : Number of medication orders in the denominator entered using CPOE.

Stage 2 Goal : 60%

Exclusion : Provider who writes fewer than 100 medication orders during the EHR reporting period.

ChARM EHR Workflow : Follow the steps below to add medication orders using CPOE.

  1. Open new encounter from 'Patient Dashboard' or 'Calendar > List View'.
  2. Choose the required Chart Type.
  3. Go to Recommendation > Rx section.
  4. Use 'Instant Add' or 'Choose from Templates' option to prescribe medications.

Laboratory Order

Denominator : Number of laboratory orders created by the Provider during the EHR reporting period

Numerator : The number of orders in the denominator recorded using CPOE.

Stage 2 Goal : 30%

Exclusion : Provider who writes fewer than 100 laboratory orders during the EHR reporting period.

ChARM EHR Workflow : Follow the steps below to add laboratory orders using CPOE.

  1. Open new encounter from 'Patient Dashboard' or 'Calendar > List View'.
  2. Choose the required Chart Type.
  3. Go to Recommendation > Lab section.

Use ‘Instant Add’ or ‘Choose from Templates’ option to add lab orders.

Radiology Order

Denominator : Number of radiology orders created by the Provider during the EHR reporting period.

Numerator : The number of orders in the denominator recorded using CPOE.

Stage 2 Goal : 30%

Exclusion : Provider who writes fewer than 100 radiology orders during the EHR reporting period.

ChARM EHR Workflow : Follow the steps below to add radiology orders using CPOE.

  1. Open new encounter from 'Patient Dashboard' or 'Calendar > List View'.
  2. Choose the required Chart Type.
  3. Go to Recommendation > Imaging section.

Use 'Instant Add' or 'Choose from Templates' option to add radiology orders.

For more information, refer to the CMS guideline for this measure
