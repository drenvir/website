Core 2 : Drug Interaction Checks

Measure Overview : Check for Drug-Drug and Drug-Allergy interactions before prescribing medications.

Exclusion : None

ChARM EHR Workflow :
This measure is satisfied by default as ChARM EHR provides Drug-Drug and Drug-Allergy interaction alerts while prescribing medications.

  1. You can configure the alert severities to be shown under Settings > Encounter > Drug Interaction section.

For more information, refer to the CMS guideline for this measure
