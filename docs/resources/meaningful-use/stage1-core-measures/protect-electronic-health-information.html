Core 13 : Protect Electronic Health Information

Measure Overview : Conduct a security risk analysis and implement security updates as necessary.

Exclusion : None

ChARM EHR Workflow :

Give high priority for protecting patient's health information. Conduct periodic security risk analysis within your practice and train your staff about HIPAA security and privacy measures.

Create and keep a documentation based on the outcome of security risk analysis.

Once this done, manually select this measure in ChARM EHR MU Dashboard.

For more information, refer to the CMS guideline for this measure
