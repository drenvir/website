Add Patients

  1. Go to Patients tab and click on "New Patient"
  2. Enter the First Name, Last Name, Date of Birth, Gender, as they are mandatory fields
  3. Provide "Email" address of the patient under the "Contact Details" section, if you have chosen "Send PHR invitation mail to Patient" as "Yes".
  4. Click on "Add Patient" to add the patient.
  5. You can also enter other details about the patients like their past medical history, active meditations, allergies, insurance information, etc., by clicking on each of those tabs at the top.
  6. If you have a list of patients to be added in bulk, you can use the "Import Patients" option.  You need to create a CSV file in the format specified, to use this option.
