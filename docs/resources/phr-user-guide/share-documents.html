Share Documents with Practice

You can share your documents with practice from ‘Documents’ section.

To share a document, click on the ‘Share File’ button, which brings a file chooser dialog.

Choose the required file from your computer.

Select the practice member(s) to be notified about file sharing.

Once confirmed, the selected members will get a notification message with link to access the shared files.

All the documents shared by you are listed under Documents > ‘Shared by me’ section.
