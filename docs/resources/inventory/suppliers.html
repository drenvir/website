Suppliers

  • Add Supplier
  • Edit Supplier Details
  • Delete Supplier

Add Supplier

  1. Navigate to inventory
  2. Click on "Supplier"
  3. Click on "Add Supplier" button to add a supplier
  4. Enter the required details and click on "Add" button to save the supplier details.

Edit Supplier Details

  1. Click on the gear icon against the supplier name and select "Edit" option to edit the supplier details
  2. Edit the required details and click on "Modify" button to save changes

Delete Supplier

  1. Click on the gear icon against the supplier name and select "Delete" option to delete the supplier
  2. Click on OK button to confirm the delete operation
