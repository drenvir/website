Reports

  1. Navigate to Inventory
  2. Click on Reports
  3. Select the report you want to view, from the listed reports
  4. Specify the date range and click on Show button to generate report
  5. If needed, Administrator can export the report either as a .pdf file or .csv file, by clicking on the required option
  6. Alternatively, search for an item or information and export the file

    Note: These operations are generic in nature and applicable on reports under inventory
