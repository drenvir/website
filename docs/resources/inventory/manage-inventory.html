Add Inventory

  1. Navigate to Inventory
  2. Click on "Add inventory"
  3. Select the "Group"
  4. Select the "Category"
  5. Search for the product by entering the product name or add a new product in the stock
  6. Enter the required stock details and click on "Save" button to add the stock to the inventory

Import Inventory

  1. Navigate to inventory
  2. Click on "Import / Export inventory" button
  3. In the pop up window select "Import" option, select the category and upload the .csv file from local storage
  4. Click on submit button to import the inventory list

    Note:

    • Maximum file size allowed is 10 MB
    • Maximum 500 entries can be imported
    • Please select a Comma Separated Values file with data arranged in below format
    • Product Name, Trade Name, Strength, Unit, Dose Form, Route, Pack, Manufacturer, Storage Location, Batch Number, Expiry Date (mm/dd/yyyy), Quantity, Reorder Level, Cost Price/Item, Retail Price/Item, Product ID, Order Quantity,Supplier Name,Tax Name,Ref. Number
    • Fields marked with are mandatory
    • To import Tax Name ensure that the Tax Name is configured under "Settings >Billing>Billing Settings>Taxes" section
    • Ensure that the file does not contain column names
  5. Search for the product by entering the product name or add a new product in the stock
  6. Enter the required stock details and click on save button to save changes

Export Inventory List

  1. Navigate to inventory
  2. Click on "Import / Export inventory" button
  3. In the pop up window select Export option, category, status and appropriate export type
  4. Click on "Export" button to import the inventory list.

    Note to select appropriate export type:

    • All Stocks : This will export all products. If a product has multiple available stock quantity, it will export all of them. If a product does not have available stock quantity it will export latest stock details
    • Available Stocks : This will export only products having available stock quantity greater than 0
    • Reorder Products : This will export all the products that reached reorder level. Pack, Manufacturer, Storage Location, Batch Number, Expiry Date (mm/dd/yyyy), Cost Price/Item and Retail Price/Item are shown from latest stock of the products

View Stock Detail

  1. Navigate to Inventory
  2. Click on the gear icon and select "Stock Details" option
  3. On the next screen complete stock history will be displayed

Edit Stock Detail

  1. Navigate into stock details
  2. Click on the gear icon and select "Edit Stock Details" option
  3. Edit the required details and click on "Update" button to save changes

Manage Damaged/Expired Stock

  1. Navigate into stock details
  2. Click on the gear icon and select "Manage Damaged / Expired Stock" option
  3. Enter the numbers of items that got damaged, expired and returned
  4. Click on "Update" button to save changes

Edit Product Details

  1. Navigate to Inventory
  2. Click on the gear icon and select "Edit Product Details" option
  3. Changing the product name is not possible, but the user can update category, product id, tax, reorder level and order quantity
  4. Click on "Modify" button to save changes

Add Reorder Level

  1. Navigate to Inventory
  2. Click on the gear icon and select "Add Reorder Level" option
  3. Enter the required details
  4. click on "Add" button to save the changes

Make Items Inactive

  1. Navigate to Inventory
  2. Click on the gear icon and select "Make Inactive" option
  3. Click OK to confirm in the pop up window

Make Items Active

  1. Navigate to Inventory
  2. Change status to inactive to view inactive items
  3. Click on the gear icon and select "Make Inactive" option
  4. Click OK to confirm in the pop up window

Delete Item from Inventory

  1. Navigate to Inventory
  2. Click on the gear icon and select "Delete InveNtory" option
  3. Click OK to confirm in the pop up window

    Note: Only active items can be deleted

Manage Inventory

  1. Click on settings~> Lab / Image / Rx~> Manage Supplements and Drugs
  2. Under "Supplements" and "Drugs" one can delete unavailable supplements and drugs
  3. For multiple record deletion use select boxes and click on "Delete" button

    OR


  4. For a single record deletion, select and delete can be used or click on "X" sign beside the product name to delete the record

    Note: Use the search box to narrow the search result
