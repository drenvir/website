Add Category

  1. Navigate to Inventory
  2. Click on "Category"
  3. Click on "Add Category" button
  4. Select appropriate "Group". Exercise caution, you cannot change the group name later.
  5. Enter the category name and description
  6. Click on "Add" button to save the category

Edit Category

  1. Click on the gear icon against the category to be edited and select "Edit" option
  2. Edit the details and click on "Modify" button to save changes

Delete Category

  1. Click on the gear icon against the category to be deleted and select "Delete" option
  2. Click on "OK" button to confirm
