Configure Procedure Codes

  1. Click on Settings~> Billing~>Procedure Codes

  2. Click on "New Code" button to add procedure code
  3. Enter the required details and click on "Add" button to add the procedure code

Edit Procedure Code

  1. Click on Settings~> Billing~>Procedure Codes
  2. Click on the gear icon and select "Edit" option to edit procedure code
  3. Edit the detail and click on Update button to save changes

Delete Procedure Code

  1. Click on Settings~> Billing~>Procedure Codes
  2. Click on the gear icon and select "Delete" option to delete procedure code
  3. Click on OK button to confirm the deletion on the pop up window

Add Procedure Category

  1. If required category is not available then navigate to "Add Procedure Code" window and click on "Add New" link to add the category
  2. Give a name to the category and click on "Add" button in the pop up window
