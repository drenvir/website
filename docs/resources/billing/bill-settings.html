Currency Settings

  1. Click on Settings ~> Billing ~> Billing Settings

  2. Under currency Setting enter the currency code and currency symbol
  3. Click on "Save" button to save changes

Invoice Settings

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under "Invoice Settings", you can do the following:
    • select the default invoice template
    • select how to display the total
    • opt to select the primary insurance by default
    • auto select Dx pointers
    • set pay invoice in full by default
    • set pay products in full by default
    • print patient signature on invoice or bill
    • show notes on invoice or bill as well as the terms and condition on the bill
    • opt to send invoice by default on patient email or PHR portal

Configuring Invoice Number Format

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under "Invoice Number Format" enter the format and starting number for the invoices
  3. Click on "Save" button to save changes

    Note: Configure invoice number format for all the facilities

Configuring Receipt Number Format

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under "Receipt Number Format" enter the format and starting number for receipts
  3. Click on "Save" button to save changes

    Note: Please configure receipt number format for all the facilities

Configuring Payment Method

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under "Payment Methods" click on "Add Payment Method" button to add a payment method

Edit Payment Method

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under "Payment Methods" click on the gear icon and select "Edit" option to edit a payment method
  3. Click on "Update" button to save changes

Delete Payment Method

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Click on the gear icon and select "Delete" option to delete a payment method

Configuring Adjustment Type

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under Adjustment Types click on "Add Adjustment Type" button to add an adjustment type

Edit Adjustment Type

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under Adjustment type click on the gear icon and select "Edit" option to edit an Adjustment type
  3. Update the details and click on Update button to save changes

Delete Adjustment Type

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under "Adjustment Types" click on the gear icon and select Delete option to delete an adjustment type

Configuring Write-Off Type

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under "Write-off Types" click on "Add Write-off Type" button to add a write-off type

Edit Write-off Type

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under "Write-off Types" click on the gear icon against the item to be edited and select "Edit" option to edit a write-off type
  3. Update the details and click on Update button to save changes

Delete Write-off Type

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under Write-off Types click on the gear icon and select Delete option to delete a write-off type

Configuring Receipt Settings

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under "Receipt Settings" select the default payment method, default adjustment type, default write-off type and auto adjust for unused payment from past receipts
  3. Click on "Save" button to save changes

Configuring Claim Status List

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under "Claim Status List" click on "Add Claim Status" button to add a Claim Status
  3. Give a name and click on "Add" button to save newly added claim status

Edit Claim Status Type

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under "Claim Status List" click on the gear icon and select Edit option to edit a claim status
  3. Update the details and click on Update button to save changes

    Note: Only manually added claim status can be deleted or edited

Delete Claim Status Type

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under "Claim Status List" click on the gear icon and select Delete option to delete a claim status

    Note: Only manually added claim status can be deleted or edited

Configuring Tax

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under "Taxes", click on "Add Tax" button to add a tax and make sure that the correct facility is selected
  3. Give a name to the Tax and percentage of Tax applicable. Click on "Add" button to save newly added Tax

Edit Tax

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under "Taxes", click on the gear icon and select "Edit" option to edit a Tax
  3. Update the details and click on "Update" button to save changes

Delete Tax

  1. Click on Settings ~> Billing ~> Billing Settings
  2. Under "Taxes", click on the gear icon and select Delete option to delete a tax
