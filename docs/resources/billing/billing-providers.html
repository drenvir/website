Configure Billing Providers

  • Add Billing Provider
  • Edit Billing Provider
  • Delete Billing Provider

Add Billing Provider

  1. Click on Settings~> Billing~> Billing Providers

  2. Click on "New Provider" button to add a Bill Provider
  3. Select the billing provider as an Organization or an Individual and enter their name accordingly.
  4. Enter required details and click on "Add" button to save billing provider

Edit Billing Provider

  1. Click on Settings~> Billing~> Billing Providers
  2. Click on the gear icon against the provider name to be edited and select "Edit" option to edit billing provider details
  3. Edit the detail and click on Update button to save changes

Delete Billing Provider

  1. Click on Settings~> Billing~> Billing Providers
  2. Click on the gear icon against the provider name to be deleted and select "Delete" option to delete billing provider
  3. Click on OK button to confirm the deletion on the pop up window
