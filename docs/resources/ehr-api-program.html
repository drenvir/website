Introduction

ChARM EHR API Program offers comprehensive set of REST APIs, that allows other medical applications and devices to interact with ChARM EHR seamlessly. If you are interested in building or integrating your application with ChARM, please fill in the form below. We will get back to you soon with instructions to get started with ChARM API. It's absolutely FREE!

API Documentation

Steps to use ChARM API

  1. Create an account with ChARM EHR, make a note of the Username
  2. Fill out the "API Access Request" form on this page
  3. We will review and send the instructions on using the API
  4. Access API documentation and build your application
